# Report Compilation

Compile a final astrophysics research report from:
- selected paper metadata
- structured paper analyses
- field synthesis
- status-labeled hypotheses (validated/plausible/rejected)

The report should include:
- executive summary
- evidence-grounded findings
- clearly separated assistant inference
- key tensions/disagreements
- hypotheses with status and rationale
- limitations and open questions

Cite source papers explicitly. Do not invent citations or constraints.
