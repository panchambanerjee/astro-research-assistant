# Astro Research Assistant (CrewAI)

Python scaffold for an astrophysics/cosmology/astronomy research assistant built with CrewAI.

## Project Goal

Build a multi-agent research workflow that can:
- discover and retrieve relevant papers,
- normalize metadata from trusted sources,
- analyze and synthesize findings into structured outputs,
- maintain an evolving research wiki and reports.

This repository currently provides a clean project scaffold only. Agent and tool implementations are intentionally not added yet.

## MVP Command

```bash
python main.py
```
